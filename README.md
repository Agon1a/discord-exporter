# Discord Exporter

Браузерное расширение (Chrome/Edge, Manifest v3) для экспорта сообщений из Discord Web в CSV по диапазону дат.

English version: [README.en.md](README.en.md)

## Возможности
- Экспорт сообщений за выбранный диапазон дат в CSV.
- Автопрокрутка вверх для подгрузки истории канала.
- Отображение открытого канала (сервер + канал) в UI расширения.
- Светлая/тёмная тема с сохранением выбора.
- Работает в доменах `discord.com`, `ptb.discord.com`, `canary.discord.com`.

## Установка (Load unpacked)
1. Открой `chrome://extensions` (или `edge://extensions`).
2. Включи **Режим разработчика**.
3. Нажми **Load unpacked** и выбери папку проекта `discord-exporter`.

## Как пользоваться
1. Открой Discord Web и перейди в нужный канал (URL вида `https://discord.com/channels/...`).
2. Открой расширение, задай даты «С» и «По».
3. Нажми **Экспортировать**. Скачается файл `discord_export_YYYY-MM-DD_to_YYYY-MM-DD.csv`.

## Структура проекта
- `public/popup.html` — UI попапа.
- `public/styles.css` — стили попапа.
- `public/icons/` — иконки расширения.
- `src/lib/popup-utils.js` — логика попапа (тема, CSV, сообщения, сбор метаданных канала).
- `src/lib/content-utils.js` — логика скролла и сбора сообщений.
- `src/popup.js` — запуск `window.DiscordExporterPopup.init()`.
- `src/content.js` — запуск `window.DiscordExporterContent.init()`.
- `manifest.json` — манифест расширения.

## Настройки экспорта
В `src/lib/content-utils.js` можно подстроить поведение:
- `stepPx` — шаг прокрутки вверх (px).
- `waitMs` — ожидание подгрузки после шага.
- `maxLoops` — максимальное число итераций.
- `noProgressLimit` — лимит попыток без прогресса.

## Ограничения и заметки
- Экспорт зависит от DOM Discord. Если Discord меняет структуру, селекторы могут перестать работать.
- Для длинной истории нужен запас времени — скролл идет вверх шагами.
- Отображение канала в UI зависит от доступных селекторов и может иногда не определяться.

## Траблшутинг
- **«Receiving end does not exist»**: обнови страницу Discord (F5) и повтори экспорт.
- **Сообщения не найдены**: проверь диапазон дат, дождись полной загрузки канала.
- **Не открывается CSV**: убедись, что браузер не блокирует загрузки.
